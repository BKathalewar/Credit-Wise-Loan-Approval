# 💳 CreditWise – Loan Approval Prediction System

## 📖 Project Overview

CreditWise is a Machine Learning-based Loan Approval Prediction System designed to predict whether a loan application should be Approved or Rejected based on an applicant's financial, personal, and credit-related information.

The primary objective of this project is to compare multiple Machine Learning classification algorithms and identify the model that best predicts loan approval outcomes.

---

## 🤖 Machine Learning Models Used

The following supervised Machine Learning algorithms were implemented and compared for loan approval prediction:

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Gaussian Naive Bayes

These models were trained on historical loan application data after performing data preprocessing, feature engineering, and exploratory data analysis.

## ✨ Features

* Predicts whether a loan application should be Approved or Rejected.
* Performs data preprocessing and handling of missing values.
* Includes Exploratory Data Analysis (EDA) to understand applicant and loan characteristics.
* Implements feature encoding and data preparation for model training.
* Compares multiple Machine Learning classification algorithms.
* Evaluates model performance using standard classification metrics.
* Demonstrates a complete end-to-end Machine Learning workflow for a real-world loan approval problem.
